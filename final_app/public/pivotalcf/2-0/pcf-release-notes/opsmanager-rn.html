New Features in Ops Manager v2.0

Configure Tiles with Runtime Configs

Tile authors can include runtime_configs as a top-level key in tile metadata to define global deployment configurations. Named runtime config settings apply to all VMs in a deployment.

Version 2.0 Alpha of Ops Manager supports defining any number of runtime_configs in an existing tile. Tile authors can also create a tile that includes a runtime config only and does not define any job types or errands.

For more information, see the Managing Runtime Configs topic in the PCF Tile Developers Guide.
